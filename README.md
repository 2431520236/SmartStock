# SmartStock

因为在15年年初被朋(sun)友忽悠了进股市,在经历了15年大跌和几次熔断之后站在了高高的山顶吹风(手动微笑).
因为我遇到喜欢的事物我就想着能不能在这方面写个app来方便下自己使用,于是就出现了这个选股app了,这里面的数据全是来自于网上,仅供参考.
